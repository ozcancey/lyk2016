Temel Web Teknolojileri

HTTP

Hypertext Transfer Protocol

Sunucu ve istemci arasındaki iletişim kurallarını belirleyen, metin tabanlı iletişim protokolüdür.

80 portunu kullanır.

SSL

Secure Sockets Layer

Sunucu ve istemci arasında güvenli veri iletişimi sağlayan protokoldür.

HTTPS

Secure Hypertext Transfer Protocol

HTTP protokolünün SSL/TLS protokolü ile birleşmesinden oluşan protokoldür.

SSL sertifikası gerektirir.

443 portunu kullanır.

HTTP Durum Kodları

HTTP durum kodları 5'e ayrılmıştır.

Kod Mesaj
1** Bilgi
2** Başarı
3** Yönlendirme
4** İstemci Hatası
5** Sunucu Hatası

Yaygın HTTP Durum Kodları

Kod Mesaj
200 OK.
201 Oluşturuldu.
301 Kalıcı olarak taşındı.
302 Geçici olarak taşındı.
400 Kötü istek.
401 Yetkisiz.
404 Bulunamadı.
500 İç sunucu hatası.

HTTP İstek Metodları

Sunucu ile veri alışverişinde bulunmak için kullanılan metodlardır.

İstek(request) sunucu tarafında işleme alınıp istemciye bir yanıt(response) gönderilir.

Response header ve body olmak üzere iki kısımdan oluşur.

GET

Sunucudan veri almak için kullanılan metoddur.

HEAD

Sunucudan bir verinin sadece header değerini almak için kullanılan metoddur.

POST

Sunucuya veri göndermek için kullanılan metoddur.

PUT

Sunucudaki bir veriyi değiştirmek için kullanılan metoddur.

DELETE

Sunucuduaki bir veriyi silmek için kullanılan metoddur.

TRACE

Sunucuya yollanan isteği yanıt olarak geri dönen metoddur.

OPTIONS

Sunucunun, istek yapılan adres için hangi metodların kullanılabileceği bilgisini dönen metoddur.

CONNECT

Sunucunun, başka bir sunucuya istek yapmasını sağlayan metoddur.

PATCH

Sunucudaki bir verinin, sadece belli bir bölümünü değiştirmek için kullanılan metoddur.

Cookie (Çerez)

Web uygulamalarının, gerektiğinde tekrar kullanmak için istemcinin bilgisayarında oluşturduğu metin dosyalarıdır.

Web Storage

Web uygulamalarının veriyi yerel olarak tutmak için kullanılan servistir. Veri kullanıcının tarayıcısına kaydedilir.

Local Storage

Veriyi sona erme tarihi olmadan depolar.

Session Storage

Veriyi tek bir oturum için depolar. Oturum sona erdiğinde veri silinir.

API

Bir uygulamanın belli işlevlerini, farklı bir uygulamanın kullanılabilmesi için oluşturulmuş arabirimdir.

Web Servis

HTTP protokolü kullanarak iki cihaz arasında iletişim sağlayan fonksiyonlar bütünüdür.
